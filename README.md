# alzt

艾力扎提的个人简历网站，可直接部署到 GitHub Pages。

## 文件说明

- `index.html`: 网站首页
- `styles.css`: 页面样式
- `resume-onepage.pdf`: 精简版一页 PDF 简历
- `assets/images/`: 头像与脱敏项目截图

## GitHub Pages 部署

仓库 Pages 设置建议：

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`

## 展示边界

项目内容只展示脱敏成果、流程方法、截图证据和能力证明，不展示账号、密码、Cookie、Token、原始答案或平台私有接口参数。
