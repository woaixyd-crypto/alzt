# 艾力扎提个人简历网站

这是一个可直接部署到 GitHub Pages 的静态个人简历网站。

## 文件说明

- `index.html`: 网站首页
- `styles.css`: 页面样式
- `resume-onepage.pdf`: 精简版一页 PDF 简历
- `assets/images/`: 头像与脱敏项目截图

## GitHub Pages 部署

1. 新建 GitHub 仓库。
2. 上传本目录下所有文件。
3. 在仓库 `Settings -> Pages` 中选择 `Deploy from a branch`。
4. 分支选择 `main`，目录选择 `/root`。
5. 保存后等待 GitHub Pages 生成访问链接。
